# Resources

Annotated reference material — books, courses, papers, and tools that inform the work in this repo. The `reading-list.md` is the primary artifact here: not a flat list, but a curated set of sources with notes on why each one matters and what it's actually useful for.

The criterion for inclusion: would I point a colleague to this and say "this changed how I think about X"? If not, it doesn't belong here.
